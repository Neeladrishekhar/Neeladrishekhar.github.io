This is the hompage for the CS 101 course being offered in Summer for the year 2019. You can find the LECTURES by following the link. Check the Announcement page for all updates regularly. Here is the link for Prutor used for the labs/practice coding. Have fun :)

Desciption from CSE website .This course provides an introduction to problem solving with computers using a modern language such as C/C++. Topics covered will include :

  • Utilization : Developer fundamentals such as editor, integrated programming environment, Unix shell, modules, libraries.
  • Programming features : Machine representation, primitive types,arrays and records, objects, expressions, control statements, iteration,procedures, functions, and basic i/o.
  • Applications : Sample problems in engineering, science, text processing, and numerical methods.
